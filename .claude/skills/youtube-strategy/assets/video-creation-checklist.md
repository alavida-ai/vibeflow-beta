# Video Creation Checklist: Full Production Quality Assurance

## Pre-Production

- [ ] Idea passes elimination criteria
- [ ] Niche validates (triple Venn + TAM)
- [ ] 30-40 titles generated, best selected
- [ ] 2-3 thumbnail variations created
- [ ] Hook script outlined (first 30 seconds)
- [ ] Full video outline/script prepared
- [ ] Content delivery plan clear

## Hook Section (First 30 Seconds)

**First 7 seconds:**
- [ ] Shows what thumbnail promised
- [ ] Same visual framing as thumbnail
- [ ] Topic immediately stated
- [ ] Viewer feels they made right click

**Next 23 seconds:**
- [ ] New curiosity loop introduced
- [ ] Pattern interrupt (visual/audio/music shift)
- [ ] Reason to keep watching established
- [ ] No retention killers present

## Body Section

- [ ] Delivers on packaging promise
- [ ] Clear progress toward payoff
- [ ] Pattern interrupts every 15-30 seconds
- [ ] Pacing appropriate (not monotonous)
- [ ] No retention killers identified
  - [ ] No "to recap..." language
  - [ ] No dead air or silence
  - [ ] No confusing tangents
  - [ ] No unexplained slow sections

## Payoff Section

- [ ] Satisfies viewer expectation
- [ ] Closes curiosity loops
- [ ] Respects viewer's time investment
- [ ] Leaves viewer satisfied

## Post-Production

- [ ] Title finalized (<60 chars)
- [ ] Thumbnail selected (glance test passed)
- [ ] Description written (useful, not keyword-stuffed)
- [ ] Tags added (5-10 relevant)
- [ ] Chapters/timestamps added
- [ ] Premiere/community post planned?
- [ ] End screens/cards directing to next video

## Final Quality Check

- [ ] Packaging promise aligns with content
- [ ] No retention killers found
- [ ] 80% coherence with channel content
- [ ] Elimination criteria still passing
- [ ] Ready for publication

## Pre-Upload Verification

- [ ] File formats correct
- [ ] Audio clear and consistent
- [ ] No technical glitches
- [ ] Thumbnail final version correct
- [ ] Title/description proofread

---

**Go through this checklist before every publish. Skip nothing.**
